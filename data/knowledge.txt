Python — мова програмування загального призначення

Python створив Гвідо ван Россум і вперше випустили у 1991 році. Назва походить від комедійного шоу «Monty Python's Flying Circus», а не від змії.

Основні риси Python:
- Простий і читабельний синтаксис, близький до псевдокоду
- Інтерпретована мова — код виконується рядок за рядком
- Динамічна типізація — тип змінної визначається під час виконання
- Велика стандартна бібліотека та активна спільнота

Python широко використовують у веб-розробці (Django, Flask), аналізі даних (pandas, NumPy), машинному навчанні (TensorFlow, PyTorch), автоматизації та науці.

Щоб встановити Python, завантажте інсталятор з python.org. Рекомендована версія — Python 3.10 або новіша. Пакети встановлюють через pip: pip install назва_пакета.

Віртуальне середовище (venv) ізолює залежності проєкту:
  python -m venv .venv
  source .venv/bin/activate   # Linux / macOS
  .venv\Scripts\activate      # Windows

RAG (Retrieval-Augmented Generation) — підхід, коли до запиту в LLM додають релевантні фрагменти документів. Модель відповідає на основі ваших даних, а не лише загальних знань з навчання.

Простий RAG без спеціальних бібліотек:
1. Розбити тексти на фрагменти (chunks)
2. Отримати ембедінги через OpenAI Embeddings API
3. Знайти найближчі фрагменти до питання (косинусна схожість)
4. Передати їх у system-повідомлення до ChatGPT

OpenAI API ключ зберігають у файлі .env, який не комітять у git. Запити надсилають через HTTPS на api.openai.com.
